# KeshavG_Smart_AC_Automation

ESP32 based AC Automation using:

- Firebase Realtime Database
- NVS Offline Backup
- NTP Time Sync
- Relay Control
- WiFi Monitoring
- Manual / Auto Modes

## Features

- Cloud timetable sync
- Offline operation from NVS
- Automatic relay control
- Real-time Firebase updates
- Schedule backup

## Hardware

- ESP32
- Relay Module
- AC Contactor
- DS3231 RTC (optional)
